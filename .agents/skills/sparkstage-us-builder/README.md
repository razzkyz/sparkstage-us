# SparkStage US Builder Skill

Expert agent for building a US market version of SparkStage booking & e-commerce platform with US payment gateways replacing DOKU payments.

## Quick Start

```
"Help me build a US version of SparkStage with Stripe payments"
```

## What This Skill Does

- Migrates DOKU payment integration to Stripe/PayPal/Square
- Adapts Indonesian e-commerce patterns to US market
- Replaces RajaOngkir shipping with USPS/FedEx/UPS
- Converts IDR currency to USD
- Updates i18n from Indonesian to English
- Provides complete architecture and code examples

## Tech Stack

- **Frontend:** Vite + React + TypeScript + Tailwind CSS
- **Backend:** Supabase Postgres + Edge Functions
- **Payment:** Stripe (primary), PayPal/Square (alternatives)
- **Shipping:** EasyPost, USPS, FedEx, UPS
- **Storage:** Cloudflare R2

## Use Cases

- "Migrate DOKU payments to Stripe"
- "Set up US shipping providers"
- "Convert currency from IDR to USD"
- "Create Stripe webhook handler"
- "Integrate EasyPost for shipping"
- "Update database for Stripe payment intents"
